自从MySQL数据库被oracle公司收购不在开源后，MySQL的作者就重新开发了一个完全兼容MySQL的数据库软件取名叫MariaDB.在MySQL的基础上加入了一些新的特性，很快得到了开源社区的支持。新版的CentOS7也使用MariaDB替代了MySQL，本文介绍在CentOS7上安装完MySQL后的一些基本配置。

0.0.1 安装MariaDB数据库

1
2

$ yum update
$ yum install -y mariadb-server

0.0.2 基本概念

  • mysqld ——服务器程序，守护进程
  • mysql ——客户端
  • 3306—–监听端口 服务器管理工具： 1、mysqlaccess 用于创建账户和设置权限 2、mysqladmin 命令行的数据库管理工具，交互式地查询服务器的状态和使用量，以及关闭服务； 3、mysqlshow 即可显示各数据库和各表的信息，又可查看服务器状态； 4、mysqldump 导出dump文件

0.0.3 修改MariaDB的配置文件

1
2
3
4
5

$ sudo vim /etc/my.cnf.d/server.cnf
[mysqld]
# 让MariaDB支持中文编码
character-set-server=utf8
collation-server=utf8_general_ci

1
2
3

$ sudo vim /etc/my.cnf.d/mysql-clients.cnf
[mysql]
default-character-set=utf8

更改编码为utf-8，加入上面的配置后便可以在数据库表中插入中文的数据。

0.0.4 启动MariaDB数据库服务

1
2

sudo systemctl start mariadb.service
sudo systemctl enable mariadb.service

0.0.5 初始化数据库

mysql_secure_installation

0.0.6 修改root用户密码

1

$ mysqladmin -u root -p flush-privileges password “xxxxx”

0.0.7 创建新的用户

使用MariaDB时不建议使用root账户进行操作，建议新建一个用户进行操作，可以保证数据库的安全性。

1

mysql -u root -p -e "GRANT ALL ON *.* TO 'vicent'@'%';"

上面新建了一个叫vicent的新账户，设置对所有的数据库都有权限，最后的%表示通过vicent这个账户可以从任何地方连接到这个MariaDB服务器。

0.0.8 查看用户信息

1

mysql -u root -p -e "SELECT User,Host FROM mysql.user;”

0.0.9 删除用户

1

> drop user 用户名@localhost;

0.0.10 修改用户密码

1

> set password for vicent@‘%’ = password(‘123’);

0.0.11 远程连接到MariaDB数据库

使用mycli连接数据库，mycli是Python实现的MySQL客户端，支持代码高亮和智能提示，用于如下命令安装mycli

1

pip3 install mycli

在另一台电脑上连接到MariaDB数据库：

1

mycli -h 192.168.22.156 -u vicent

学习MariaDB数据库，建议先看看《SQL基础教程》这本书学习SQL语句。本书是一本非常适合初学者入门SQL语句的书。
